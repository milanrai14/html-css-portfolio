# Portfolio Website

A personal **portfolio website** showcasing projects, blog cards, and hero section. Built using **HTML and CSS**.  

---

## Features

- **Hero Section:** Catchy headline with an image to introduce the website.
- **Highlight Cards:** Sections with images, descriptions, dates, and “Read More” buttons.
- **Blog Cards:** Multiple card layout to showcase articles or posts.
- **Responsive Layout:** Structured for easy styling and can be made responsive with CSS.
- **Footer:** Simple footer with copyright information.
